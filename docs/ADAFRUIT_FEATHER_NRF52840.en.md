# Adafruit Feather nRF52840 Express

This document describes the support for the Adafruit Feather nRF52840.

## Overview
The Feather nRF52840 features a Nordic nRF52840 SoC (ARM Cortex-M4F) with Bluetooth LE support.

## Configuration
The HAL uses the nRF52's PPI (Programmable Peripheral Interconnect) system to link timers and ADC for low-latency control without CPU intervention.
