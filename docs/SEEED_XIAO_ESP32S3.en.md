# Seeed Studio XIAO ESP32-S3

This document describes the support for the Seeed Studio XIAO ESP32-S3.

## Overview
The XIAO ESP32-S3 features a powerful dual-core ESP32-S3 chip, capable of high-performance tasks and AI applications.

## Configuration
The HAL uses the ESP32-S3's MCPWM and ADC DMA for motor control, offering similar capabilities to the generic ESP32 but with updated drivers.
