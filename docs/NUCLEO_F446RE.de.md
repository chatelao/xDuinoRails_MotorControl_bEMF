# STM32 Nucleo F446RE

Dieses Dokument beschreibt die Unterstützung für das STM32 Nucleo F446RE Board.

## Überblick
Das Nucleo F446RE verfügt über eine leistungsstarke STM32F446RE MCU (ARM Cortex-M4).

## Konfiguration
Der HAL nutzt die STM32-Hardware-Timer und den ADC für eine effiziente Motorsteuerung.
