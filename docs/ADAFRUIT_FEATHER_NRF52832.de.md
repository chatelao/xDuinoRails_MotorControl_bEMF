# Adafruit Feather nRF52832

Dieses Dokument beschreibt die Unterstützung für den Adafruit Feather nRF52832.

## Überblick
Der Feather nRF52832 verfügt über ein Nordic nRF52832 SoC.

## Konfiguration
Ähnlich wie beim nRF52840 nutzt diese Implementierung das PPI-System für eine effiziente Hardware-Abstraktion.
