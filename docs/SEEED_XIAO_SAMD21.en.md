# Seeed Studio XIAO SAMD21

This document describes the support for the Seeed Studio XIAO SAMD21.

## Overview
The XIAO SAMD21 is a tiny board based on the Atmel SAMD21G18 (ARM Cortex-M0+).

## Configuration
The HAL implementation utilizes the SAMD21's timers and ADC. Due to hardware limitations compared to the RP2040, some advanced features might have different performance characteristics.
