# Arduino Uno

Dieses Dokument beschreibt die Unterstützung für den klassischen Arduino Uno (ATmega328P).

## Überblick
Der Arduino Uno ist die Referenzplattform für das Arduino-Ökosystem.

## Konfiguration
Aufgrund der 8-Bit-Architektur und begrenzten Ressourcen ist die HAL-Implementierung für die AVR-Architektur möglicherweise weniger leistungsfähig als auf 32-Bit-ARM-Plattformen. Sie dient als Basis für Kompatibilität.
